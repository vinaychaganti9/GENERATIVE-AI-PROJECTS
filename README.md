 Project 2: GenAI Semantic Similarity 
Project Using Sentence Transformers
Project Overview
This project implements a semantic similarity analysis system using the Sentence Transformers

calculates their similarity using cosine similarity.
Objective
To understand how transformer-based sentence embeddings can measure semantic meaning 
beyond keyword matching. Technologies and Tools
